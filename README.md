# Browser AI Agent

An AI-powered Chrome extension for browser automation using natural language commands. This project leverages the browser-use Python library to execute browser tasks based on natural language instructions.

## Features

- Natural language task processing
- Real-time browser automation
- Chat-like interface for task history
- Secure communication between extension and backend
- Task progress monitoring and status updates

## Project Structure

```
├── extension/           # Chrome extension files
│   ├── manifest.json    # Extension manifest
│   ├── popup.html      # Extension popup interface
│   ├── styles/         # CSS styles
│   └── js/            # JavaScript files
├── backend/           # Python backend
│   ├── main.py        # FastAPI server
│   └── requirements.txt # Python dependencies
└── README.md         # Project documentation
```
